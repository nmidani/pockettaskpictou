Continue building the PocketTask web application with the following features and requirements.

PocketTask is a hyper-local microtask marketplace for Pictou County, Nova Scotia where neighbours can post small tasks and students or locals can accept them to earn extra money.

The system must remain simple, trustworthy, and mobile-friendly.

---

## USER ACCOUNTS

Implement authentication with two types of users:

1. Task Giver
2. Task Taker

During signup require:

* Name
* Phone number (simulate SMS verification)
* Town selection within Pictou County
* Role selection (Task Giver or Task Taker)

Available towns:

New Glasgow
Stellarton
Trenton
Westville
Pictou
River John
Abercrombie
Scotsburn

After login users should see a dashboard with nearby tasks.

User profiles must display:

* Name
* Town
* Rating
* Tasks completed
* Reviews

---

## TASK POSTING SYSTEM

Allow Task Givers to create tasks.

Task fields:

* Task title
* Description
* Payment amount
* Estimated duration
* Payment method (Cash or eTransfer)
* Location (auto detected from map)
* Category

Task categories:

Yard work
Cleaning
Moving help
Dog walking
Grocery pickup
Tech help
Small repairs
Other

After posting a task it should appear:

* On the map
* In the task list

Each task card must display:

* Title
* Payment
* Distance from user
* Town
* Apply button

---

## FIRST-COME FIRST-SERVE TASK CLAIMING

Implement task claiming rules:

* Tasks are visible to nearby Task Takers.
* The first user who clicks "Apply for Task" automatically gets the job.
* Once accepted the task becomes locked.
* Other users should see the message:

"This task has already been taken."

When a task is accepted:

* Open a chat window between the Task Giver and Task Taker.

---

## MAP WITH TASKS

Create an interactive map using Mapbox.

Requirements:

* Show tasks as map pins
* Detect the user's location
* Only display tasks within Pictou County

Clicking a map pin should show:

Task title
Payment
Distance
Apply button

Add map filters:

All tasks
Nearby tasks
My accepted tasks

---

## GEOGRAPHIC RESTRICTION

Restrict the platform to Pictou County.

If a user outside the region tries to access the platform display the message:

"PocketTask currently operates only in Pictou County. Expansion coming soon."

Use geolocation coordinates to enforce the restriction.

---

## RATING SYSTEM

After task completion both users rate each other.

Rating scale: 1 to 5 stars.

User profiles must show:

* Average rating
* Number of completed tasks
* Reviews

---

## SAFETY AND REPORTING

Add a report system.

Each profile and task should have a Report button.

Report reasons:

Did not show up
Scam
Harassment
Unsafe task
Other

Create a simple Community Guidelines page with the rules:

1. Be respectful.
2. Only post safe and legal tasks.
3. Honor task commitments.
4. Pay agreed amounts.
5. No illegal or dangerous activities.
